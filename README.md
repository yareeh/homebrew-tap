# Homebrew Tap

Homebrew formulae for yareeh CLI tools.

## Usage

```bash
brew tap yareeh/tap
brew install themoviedb-cli
brew install goodreads-cli
```

Or install directly:

```bash
brew install yareeh/tap/themoviedb-cli
brew install yareeh/tap/goodreads-cli
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [themoviedb-cli](https://github.com/yareeh/themoviedb-cli) | CLI for The Movie Database (TMDB) |
| [goodreads-cli](https://github.com/yareeh/goodreads-cli) | CLI for Goodreads with browser automation |
